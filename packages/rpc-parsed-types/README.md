# @trezoa/rpc-parsed-types

This package defines types for parsed objects returned by methods of the [Trezoa JSON-RPC](https://docs.trezoa.com/api/http). It can be used standalone, but it is also exported as part of Kit [`@trezoa/kit`](https://github.com/trezoa-xyz/kit/tree/main/packages/kit).
